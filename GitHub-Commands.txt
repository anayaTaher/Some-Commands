==========> GitHub Commands <==========

=====> To Push Your Files Into Github <=====
    => Init  : git init
    => Clone : git clone "Repository URL"
    => Add   : git add *
    => Commit: git commit -m "Your Description Message"
    => Push  : git push origin main

=====> To Pull Your Files From Github <=====
    => Pull: git pull origin

=====> To Show Your Files Status <=====
    => Show Status: git status
